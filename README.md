📦 Installation

1. Cloner le projet

git clone https://github.com/valerylusty-glitch/signal-ghost-md.git
cd signal-ghost-md

2. Installer les dépendances

npm install

3. Lancer le bot

node index.js


---

🔐 Configuration

📱 Connexion WhatsApp

Lance le bot

Scan le QR code avec WhatsApp

---

📜 Commandes de base

Commande	Description

!menu	Affiche le menu
!ai	Pose une question à l'IA
!ban	Bannir un membre
!kick	Expulser un membre
!xp	Voir ton XP
